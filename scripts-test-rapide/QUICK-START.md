# ⚡ Quick Start - Tests Rapides

## 🚀 Pour Tester Rapidement

```powershell
cd scripts-test-rapide
.\test-rapide.ps1
```

Choisir :
- Mode 1 (Development) pour itérer rapidement
- Mode 2 (Production) pour validation finale

## 🔄 Pour Déployer et Vérifier

```powershell
cd scripts-test-rapide
.\deployer-et-verifier.ps1
```

Le script fait tout automatiquement et vous dit si Local = Render.

## ✅ C'est Tout !

**Développement :** `test-rapide.ps1` → Mode 1  
**Validation :** `test-rapide.ps1` → Mode 2  
**Déploiement :** `deployer-et-verifier.ps1`
